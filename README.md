2. Install Node dependencies and build Tailwind CSS (from the `gearguard` folder):

```powershell
cd gearguard
npm install
npm run build:css        # build once
npm run watch:css        # development watch (optional)
cd ..
```

3. Apply migrations and create a superuser:

```powershell
python gearguard/manage.py migrate
# If the management command exists (project references a helper):
python gearguard/manage.py create_initial_superuser
# otherwise use the standard Django command:
python gearguard/manage.py createsuperuser
```

4. Run the development server:

```powershell
python gearguard/manage.py runserver
```

Open http://127.0.0.1:8000/ to view the app.

> - Static files for production should be collected and served by your web server.
